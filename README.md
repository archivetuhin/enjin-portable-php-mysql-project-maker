- **Fully Portable:** Run from USB or any folder. No installation required.

## âš¡ Quick Start

1. **Download & Extract:** Extract to `G:\enjin\`.
2. **Create Project:** Inside `root\`, create your project folder (e.g., `my_project`).
3. **Run Server:** Double-click `run.bat` and follow the prompts:
   - Choose project folder
   - Select PHP version
   - Enable SSL if needed
   - Set custom host/port (optional)

Your project will start with **PHP FastCGI + nginx + SSL** automatically.
