About Enjin

Enjin is a portable Windows development environment designed for web developers who want a fast, lightweight, and self-contained solution. It includes PHP, MySQL, and Nginx, allowing you to run and test web projects locally without installing multiple tools or modifying system settings.

With Enjin, you can create projects, manage servers, enable SSL, and switch PHP versions with a single command. It’s perfect for local web development, testing PHP applications, and experimenting with frameworks like Laravel or CodeIgniter on Windows.

Built for portability and automation, Enjin simplifies your workflow by managing services, automating repetitive tasks, and providing a reliable local PHP stack that works anywhere. Whether you are a beginner learning web development or an experienced developer testing production-ready projects, Enjin delivers a fast and efficient Windows-based web development environment.

User Guide

Follow these steps to set up your portable environment.

1

Download & Setup

Download from Git or clone repository. If you downloaded ZIP file, unzip it and move to folder to your preferred drive.

2

Launch Application

Locate project root folder and double-click on run.bat. The CMD window will open to guide you.

3

Project Folder Name

The prompt will ask you for your project folder name. Enjin creates a dedicated root folder where all your PHP projects will be stored.

C:\Windows\System32\cmd.exe
Microsoft Windows [Version 10.0.19045.3693]
(c) Microsoft Corporation. All rights reserved.

C:\Enjin> Please enter your project folder name:
4

Existing or New Project?

Choose to continue with existing folder or create a new project folder inside root directory.

C:\Windows\System32\cmd.exe
C:\Enjin> Folder found. Continue with existing?
1. Yes
2. No
5

Local Domain Setup

If you want to use a local domain (e.g., enjin.local), Enjin will automatically create an entry in your Hosts file.

C:\Windows\System32\cmd.exe
C:\Enjin> Use local domain?
1. Yes
2. No
6

Select PHP Version

Select PHP version required for your specific project.

C:\Windows\System32\cmd.exe
Select PHP version (7.4, 8.0, 8.1, 8.3):
1. php7.4
2. php8.0
3. php8.1
4. php8.3
Enter Choice:
7

Composer Integration

Select if you need Composer for dependency management.

C:\Windows\System32\cmd.exe
C:\Enjin> Do you need Composer for this project?
1. Yes
2. No
8

SSL Configuration

Decide if you need SSL. Enjin will configure certificates for HTTPS if you choose yes.

C:\Windows\System32\cmd.exe
C:\Enjin> Do you need SSL?
1. Yes
2. No

Manual Installation Guide

How to manually install PHP or MySQL if you need specific versions not included by default.

Installing Multiple PHP Versions

1. Download PHP Non-Thread Safe (NTS) zip archive for Windows from php.net.

2. Extract contents to Enjin library folder:

C:\enjin\lib\php\php81 C:\enjin\lib\php\php84

3. Rename folder to match version (e.g., php{version}).

4. Restart run.bat; new version will appear in selection menu.

Installing MySQL / MariaDB

1. Download MySQL or MariaDB zip archive (no installer) for Windows.

2. Extract contents to Enjin library folder:

C:\enjin\lib\mysql\mysql90 C:\enjin\lib\mysql\mariadb11

3. Rename folder to mysql{v-no}.

4. Enjin scripts will automatically detect bin\mysqld.exe inside these folders.

Frequently Asked Questions

Is Enjin really free? +
Yes, Enjin is completely free and open-source under MIT License. You can use it for personal and commercial projects.
Can I run multiple projects at once? +
Currently, Enjin runs one instance of server at a time via interactive command line menu. However, you can easily switch between projects instantly using run.bat menu.
Does it require Administrator rights? +
Generally no, since it is portable. However, if you choose to set up a local domain (editing Windows Hosts file), script may prompt you for Administrator privileges.
Where is my project data stored? +
All your project files are stored in root\ folder inside Enjin directory. Database data is stored inside lib\mysql\mysql{version}\data\. Moving Enjin folder moves your data.
