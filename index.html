Why Enjin?

Perfect for testing legacy and modern apps side-by-side.

Fully Portable

Put Enjin on a USB stick or cloud folder. Your entire environment travels with you.

Multi-PHP Support

Run PHP 7.4, 8.0, and 8.1 simultaneously on different ports.

Clean Structure

Manual project control. Create folders in `root` and use to interactive runner.

Folder Structure

G:\enjin\
├─ php\
│ ├─ php81\
│ ├─ php82\
│ └─ ...
├─ root\ # (project folders created here)
├─ database\
│ ├─ mysql57\
│ ├─ mysql80\
├─ run.bat # (Interactive Runner)
└─ config.ini

How to Use

Manual project creation followed by interactive execution.

1

Create Project Folder Manually

Open to G:\enjin\root\ folder. Create a new folder for your project (e.g., my_app). Add your PHP files (like index.php) inside it.

G:\enjin\root\my_app\index.php
2

Run to Interactive Script

Go back to main G:\enjin\ folder and double-click run.bat.

The script will ask for to folder name you just created.
It will then ask you to select a PHP version and optionally a custom port.
3

Server Starts Automatically

Once you select your options, Enjin configures to environment and starts to PHP server for you automatically.

http://localhost:8000

Configuration & Setup

Edit these files to change defaults.

config.ini

Edit this file to set your server paths. run.bat reads these settings automatically.

SERVER_ROOT=G:\enjin\
SERVER_DOC_ROOT=G:\enjin\root\
DEFAULT_HOST=localhost
DEFAULT_PORT=8000

MySQL Setup

  1. Download ZIP portable MySQL.
  2. Extract to G:\enjin\database\mysql80.
  3. Create a data folder inside.
  4. Run PowerShell as Admin to initialize:
    5. mysqld --initialize-insecure --datadir=...
  5. Start server manually.
